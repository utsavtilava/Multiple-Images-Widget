Multiple Images Widget is Widgets base plugin in which user just need to assign Sidebar to show as Site Sidebar
== Description ==

### Overview
Multiple Images Widget is allows the end user to upload a image, add title, add image link, add link new tab and multiple add all section.Easily add your sidebar.

* Add Title

* Add Image Link

* Select Open New Tab

* Upload Your Image

* Add New Image Section

== Installation ==

= Using The WordPress Widgets =

1. Navigate to the \'Add New\' in the plugins Widgets
2. Search for \'Multiple Images Widget\'
3. Click \'Install Now\'
4. Activate the plugin on the Plugin Widgets

= Uploading in WordPress Widgets =

1. Navigate to the \'Add New\' in the plugins Widgets
2. Navigate to the \'Upload\' area
3. Select `multiple-images-widget.zip` from your computer
4. Click \'Install Now\'
5. Activate the plugin in the Plugin Widgets
